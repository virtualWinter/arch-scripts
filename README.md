# arch-skripts
Arch script install / post-install

You still need a usb/dvd with the newest arch installer 

owo

# How 2 use

Install git
```
pacman -S git
```

Clone dis repo

```
git clone https://github.com/vWinter-Codes/arch-scripts.git
```
Run the install script
```
./install.sh
```
Run the post install script
```
./post-install.sh
```
